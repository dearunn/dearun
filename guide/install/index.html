安装

  • 请按照以下步骤运行Dearun Tools：
  1. 点击Dearun_Setup.exe文件
  2. 请跟随安装步骤来即可，建议安装在D、E盘等非系统盘
  3. 双击运行桌面上的Dearun.exe

如果您的电脑安装了360软件，由于软件没有数字签名，所以在使用过程中可能会被误判，但是请放心，软件安全无毒，请将Dearun.exe添加为信任文件
